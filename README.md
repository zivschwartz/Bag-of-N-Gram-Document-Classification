# Bag-of-N-Gram-Document-Classification
Bag-of-N-Gram-Document-Classification built for Natural Language Processing (DS-GA-1011).

This assignment deals with the  large movie review dataset from IMDB for binary sentiment classification. Within this large movie review dataset, there are 25,000 reviews designated for training and 25,000 designated for testing.
